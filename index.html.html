Grafika SVG

SVG

SVG (Scalable Vector Graphics) to format grafiki wektorowej oparty na XML, który skaluje się bez utraty jakości.

SVG — przykłady
<svg width="100" height="100" viewBox="0 0 100 100"
     xmlns="http://www.w3.org/2000/svg">

  Koło
  <circle cx="50" cy="50" r="40"
           fill="#4BB8A0" stroke="white" stroke-width="2"/>

  Prostokąt
  <rect x="10" y="10" width="80" height="50"
        rx="8" fill="#E8A020"/>

  Linia
  <line x1="0" y1="50" x2="100" y2="50"
        stroke="#C94F4F" stroke-width="2"/>

  Tekst
  <text x="50" y="55" text-anchor="middle"
        font-size="12" fill="white">SVG</text>

</svg>
ElementOpis
<circle>Koło — atrybuty: cx, cy, r
<rect>Prostokąt — x, y, width, height, rx (zaokrąglenie)
<line>Linia — x1, y1, x2, y2
<polygon>Wielokąt — points="x1,y1 x2,y2 …"
<path>Dowolna ścieżka — atrybut d (M, L, C, Q, Z…)
<text>Tekst w SVG — x, y, font-size, fill
<defs>Definicje: gradienty, markery, filtry

Pseudo klasy i elementy

CSS

Pseudo klasy (:)

  • :first-child — pierwsze dziecko
  • :last-child — ostatnie dziecko
  • :nth-child(n) — n-te dziecko
  • :hover — po najechaniu
  • :focus — po skupieniu
  • :active — podczas klikania
  • :visited — odwiedzone linki
  • :not(sel) — negacja

Pseudo elementy (::)

p::before {
  content: '»';
  color: #E8A020;
}
p::after {
  content: '';
}
p::first-line  { font-weight: bold; }
p::first-letter{ font-size: 2em; }
::selection    { background: #E8A020; }

Mapy odsyłaczy

HTML

Mapy odsyłaczy pozwalają definiować klikalny obszar na obrazie poprzez znacznik <map> i <area>.

<img src="mapa.jpg" alt="Mapa"
     usemap="#przykladowa-mapa"
     width="400" height="300">

<map name="przykladowa-mapa">
  Prostokąt
  <area shape="rect"
        coords="10,10,100,80"
        href="strona1.html"
        alt="Obszar 1">

  Koło
  <area shape="circle"
        coords="200,150,50"  cx,cy,r
        href="strona2.html"
        alt="Obszar 2">

  Wielokąt
  <area shape="poly"
        coords="300,10, 390,80, 350,150"
        href="strona3.html"
        alt="Obszar 3">
</map>
